# BagusBag Jakarta Website

Website katalog produk untuk jualan goodie bag, paper bag, dan tas spunbond.

## Fitur yang sudah dibuat
- Landing page SEO-friendly
- Katalog 160 produk contoh
- Search, filter kategori, filter ukuran, dan sorting harga
- Price list
- Halaman dashboard shell
- Desain responsif untuk mobile dan desktop

## Catatan penting
Versi GitHub Pages ini adalah website statis. Agar perubahan foto, harga, dan produk dari dashboard tampil permanen untuk semua pengunjung tanpa buka GitHub, website perlu disambungkan ke database dan storage seperti Supabase atau Firebase.

## GitHub Pages
Aktifkan melalui Settings > Pages > Deploy from branch > main > root.
